# NVIDIA Isaac Sim کو ترتیب دینا

**NVIDIA Isaac Sim** ایک روبوٹکس سمولیشن ٹول کٹ ہے جو NVIDIA Omniverse پلیٹ فارم پر بنائی گئی ہے۔ یہ روبوٹس کو تیار کرنے، جانچنے اور تربیت دینے کے لیے ایک فوٹوریئلسٹک، فزکس کے لحاظ سے درست اور AI-ایکسلریٹڈ ماحول فراہم کرتا ہے۔

Gazebo کے برعکس، جو تیز فزکس کو ترجیح دیتا ہے، Isaac Sim حقیقت پسندی اور NVIDIA کے AI اسٹیک (جیسے reinforcement learning کے لیے Isaac Gym) کے ساتھ انضمام کو ترجیح دیتا ہے۔

## سسٹم کی ضروریات

Isaac Sim کو ہارڈ ویئر کی سخت ضروریات درکار ہیں۔ یقینی بنائیں کہ آپ کا ورک سٹیشن کم از کم تصریحات پر پورا اترتا ہے:
-   **OS:** Ubuntu 22.04
-   **GPU:** NVIDIA RTX 2070 یا اس سے زیادہ (RTX 4070+ تجویز کردہ)۔ آپ کو آفیشل NVIDIA پروپرائٹری ڈرائیورز استعمال کرنے ہوں گے۔
-   **CPU:** Intel Core i7 یا AMD Ryzen 7 (کم از کم 6 کور)۔
-   **RAM:** 16GB (32GB+ تجویز کردہ)۔

## تنصیب کے مراحل

1.  **NVIDIA Omniverse Launcher انسٹال کریں:**
    -   [NVIDIA Omniverse ویب سائٹ](https://www.nvidia.com/en-us/omniverse/) پر جائیں۔
    -   Omniverse Launcher ڈاؤن لوڈ اور انسٹال کریں۔ یہ ایک ایسی ایپلیکیشن ہے جو Omniverse ایپس بشمول Isaac Sim کا انتظام کرتی ہے۔

2.  **لانچر سے Isaac Sim انسٹال کریں:**
    -   Omniverse Launcher کھولیں۔
    -   "Exchange" ٹیب پر جائیں۔
    -   "Isaac Sim" تلاش کریں اور اسے انسٹال کرنے کے لیے کلک کریں۔ یہ سمیلیٹر ڈاؤن لوڈ کرے گا، جو کہ بہت بڑا ڈاؤن لوڈ ہو سکتا ہے (5GB+)۔

3.  **Isaac Sim کا Python ماحول انسٹال کریں:**
    -   Isaac Sim اپنے ایمبیڈڈ Python ماحول کے ساتھ آتا ہے۔ آپ کو ایک علیحدہ ورچوئل ماحول بنانے کی ضرورت نہیں ہے۔
    -   انسٹال ہونے کے بعد، آپ فراہم کردہ `python.sh` ریپر کا استعمال کرتے ہوئے Python اسکرپٹس چلا سکتے ہیں، جو اس بات کو یقینی بناتا ہے کہ تمام ضروری ماحولیاتی متغیرات اور راستے درست طریقے سے سیٹ ہیں۔

    ```bash
    # Example of running a standalone Isaac Sim python script
    ./python.sh my_robot_script.py
    ```

4.  **ROS 2 برج کو فعال کریں:**
    -   Isaac Sim میں ROS 2 کے لیے بلٹ ان سپورٹ موجود ہے۔ اس کا انتظام ایک ایکسٹینشن کے ذریعے کیا جاتا ہے۔
    -   Isaac Sim کے اندر، `Window > Extensions` پر جائیں۔
    -   `omni.isaac.ros2_bridge` تلاش کریں اور اسے فعال کریں۔
    -   یہ خود بخود ROS 2 ٹاپکس، سروسز اور ایکشنز کو سمولیشن ماحول میں برج کر دے گا، جس سے آپ کے بیرونی ROS 2 Nodes کو سمولیٹڈ روبوٹ کے ساتھ بات چیت کرنے کی اجازت ملے گی۔

ایک بار سیٹ اپ ہونے کے بعد، آپ اپنے روبوٹ کی URDF یا USD (Universal Scene Description) فائل امپورٹ کرنا، ایک منظر ترتیب دینا اور اپنے AI کنٹرول ایجنٹس کو جوڑنا شروع کر سکتے ہیں۔
