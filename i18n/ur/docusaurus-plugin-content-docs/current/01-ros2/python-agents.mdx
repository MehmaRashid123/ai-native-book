# Python Agents with `rclpy`

`rclpy` آفیشل Python client library ہے ROS 2 کے لیے۔ یہ آپ کو ROS 2 نوڈز، پبلشرز، سبسکرائبرز، اور بہت کچھ لکھنے کی اجازت دیتا ہے، براہ راست Python میں۔ یہ بنیادی طریقہ ہے جس سے ہم اپنے AI ماڈلز کو روبوٹ کے ہارڈ ویئر کے ساتھ جوڑیں گے۔

## Writing a Simple Publisher

ایک پبلشر نوڈ ایک ٹاپک پر پیغامات بھیجتا ہے۔ آئیے ایک نوڈ بناتے ہیں جو "Hello World" پیغام شائع کرتا ہے۔

### The Code (`simple_publisher.py`)

```python
```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class SimplePublisher(Node):
    def __init__(self):
        super().__init__('simple_publisher')
        self.publisher_ = self.create_publisher(String, 'chatter', 10)
        timer_period = 0.5  # seconds
        self.timer = self.create_timer(timer_period, self.timer_callback)
        self.i = 0

    def timer_callback(self):
        msg = String()
        msg.data = f'Hello World: {self.i}'
        self.publisher_.publish(msg)
        self.get_logger().info(f'Publishing: "{msg.data}"')
        self.i += 1

def main(args=None):
    rclpy.init(args=args)
    simple_publisher = SimplePublisher()
    rclpy.spin(simple_publisher)
    simple_publisher.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```
```

### Breakdown:
1. ہم ضروری لائبریریز امپورٹ کرتے ہیں، بشمول `Node` اور `String` میسج ٹائپ۔
2. ہماری `SimplePublisher` کلاس `Node` سے وراثت میں ملتی ہے۔
3. کنسٹرکٹر میں، ہم `create_publisher()` کو کال کرتے ہیں ایک پبلشر بنانے کے لیے جو `String` پیغامات بھیجتا ہے `chatter` نامی ٹاپک پر۔
4. ہم ایک ٹائمر بناتے ہیں جو ہر 0.5 سیکنڈ میں `timer_callback` کو کال کرتا ہے۔
5. `timer_callback` فنکشن ایک `String` پیغام بناتا ہے، اسے پُر کرتا ہے، اور اسے شائع کرتا ہے۔

## Writing a Simple Subscriber

ایک سبسکرائبر نوڈ ایک ٹاپک پر پیغامات کے لیے سنتا ہے۔ آئیے ایک نوڈ بناتے ہیں جو ہمارے `chatter` ٹاپک کو سنتا ہے۔

### The Code (`simple_subscriber.py`)

```python
```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String

class SimpleSubscriber(Node):
    def __init__(self):
        super().__init__('simple_subscriber')
        self.subscription = self.create_subscription(
            String,
            'chatter',
            self.listener_callback,
            10)
        self.subscription  # prevent unused variable warning

    def listener_callback(self, msg):
        self.get_logger().info(f'I heard: "{msg.data}"')

def main(args=None):
    rclpy.init(args=args)
    simple_subscriber = SimpleSubscriber()
    rclpy.spin(simple_subscriber)
    simple_subscriber.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```
```

### Breakdown:
1. ہم `simple_subscriber` نامی ایک نوڈ کو انیشلائز کرتے ہیں۔
2. ہم `chatter` ٹاپک کو سبسکرائب کرنے کے لیے `create_subscription()` کو کال کرتے ہیں۔
3. ہم ROS 2 کو بتاتے ہیں کہ جب بھی کوئی نیا پیغام آئے تو ہمارے `listener_callback` فنکشن کو کال کریں۔
4. `listener_callback` فنکشن صرف اس ڈیٹا کو کنسول پر لاگ کرتا ہے جو اسے موصول ہوا ہے۔

`rclpy.spin()` وہ جادو ہے جو نوڈ کو چلتا رہتا ہے اور کال بیکس پر کارروائی کرتا رہتا ہے جب تک کہ آپ اسے بند نہ کردیں (مثال کے طور پر، Ctrl+C کے ساتھ)۔
